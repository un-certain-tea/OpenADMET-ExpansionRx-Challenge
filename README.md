# OpenADMET-ExpansionRx-Challenge
Entry for the OpenADMET ExpansionRx Challenge



*   **Generated Features:**
    *   **RDKit:** 2D descriptors, ECFP4 fingerprints (binary & count), 3D descriptors (WHIM, GETAWAY, RDF, MORSE, AUTOCORR3D), Avalon substructure fingerprints, Extended Reduced Graph fingerprints.
    *   **Jazzy:** (Optional) Interaction energies (delta interaction, delta polar/apolar).
    *   **ChemAxon:** (Optional) pKa, logP, logD, H/L/B.
    *   **CheMeleon:** (Optional) Deep learning embeddings.
    *   **Minimol:** Conformer-aware deep learning embeddings.
